### Flask Portfolio Website

A personal portfolio web application built using **Python Flask**, **HTML**, and **CSS**.  
This project demonstrates routing, template rendering, static file handling, and form submission using the Flask framework.

---

### Project Overview

This portfolio website includes:

-  Home Page
-  Contact Page with Form
-  Custom CSS Styling
-  Flask Routing


### Technologies Used

- Python
- Flask
- HTML5
- CSS3
